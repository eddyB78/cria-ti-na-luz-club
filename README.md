# ✦ Cria Ti Na Luz Club

**Plataforma premium de evolução espiritual** — Umbanda, Tarot, Meditação e Autoconhecimento.

> Um "Netflix espiritual" com aulas ao vivo, trilhas por objetivo e oráculo com IA.

## 🚀 Demo

👉 **[Acesse o app](https://eddyb78.github.io/cria-ti-na-luz-club/)**

## 📱 Funcionalidades

- **Home** — Dashboard inteligente com streak, carta do dia e agenda
- **Aulas ao Vivo** — Streaming com replay de desenvolvimento mediúnico, rituais e meditações
- **Trilhas** — Jornadas completas por objetivo (espiritual, emocional, financeiro, amor próprio)
- **Oráculo** — Tiragem interativa de cartas com interpretação
- **Perfil** — Gamificação com níveis, desafios e progresso
- **Comunidade** — Feed social para compartilhar evolução

## 💰 Modelo

| Plano | Preço | Acesso |

| VIP | R$ 79,90/mês | Tudo + mentoria + rituais avançados exclusivos |

## 🛠️ Stack Planejado

- **Mobile**: React Native (Expo)
- **Backend**: Supabase (PostgreSQL + Auth + Realtime)
- **Streaming**: Mux
- **IA**: Claude API (interpretação de tarot)
- **Pagamento**: Stripe + Pagar.me (PIX)
